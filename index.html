LVK news

Моя первая попытка сделать что-то серьезное. Приступая к разработке, не имел графического макета, знаний PHP и углубленных знаний front-технологий. За пример дизайна брал сайт новостей "Медуза". В результате получился новостной сайт, развернутый на CMS Modx.

Реализовал публикацию новостей общей лентой на главной и отдельно для каждой категории по тематике. При публикации новости есть возможность добавить видео c youtube. В самой новости есть теги, имя автора публикующего статью, дату публикации, количество просмотров статьи, кнопки репоста новости в соц-сети. Новости на сайте можно искать по поиску или через теги. С сайта можно написать на почту владельцам ресурса через форму обратной связи.
